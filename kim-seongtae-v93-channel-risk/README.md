# 김성태전: 제국의 시대 v9.3

이번 회차는 기존 가격 최적화 기능을 반복하지 않고 다음 세 영역을 확장했습니다.

1. 브랜드별 판매 채널 전략: DTC, B2B, 혼합, 지역 채널을 선택하고 채널 충돌 수치가 매출과 현금회수에 영향을 줍니다.
2. 계약금·미수금 통제: 선금 비율, 잔금, 회수기한, 거래처 위험도를 관리합니다.
3. 체크섬 저장 복구: v9.3 본 저장과 순환 백업 2개를 분리하고 v9.2 핵심 재무 수치를 한 번 마이그레이션합니다.
